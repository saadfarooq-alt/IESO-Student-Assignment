# IESO 2025 Intertie & Demand Analysis

Analysis of Ontario's electricity trading position using publicly available IESO datasets.

## Datasets

Both datasets are sourced from the [IESO Public Reports](https://reports-public.ieso.ca/public/) directory and stored in `datasets/`.

| File | Description |
|------|-------------|
| `PUB_IntertieScheduleFlowYear_2025.csv` | Hourly intertie imports, exports, and flow across all Ontario interconnections (Manitoba, Michigan, Minnesota, New York, Quebec) |
| `PUB_Demand_2025.csv` | Hourly Ontario electricity demand (MW) |
| `merged_2025.csv` | Cleaned and merged dataset — generated by `ieso_data_prep.py` |

## How to Run

**1. Install dependencies**
```bash
pip install pandas matplotlib jupyter
```

**2. Generate the merged dataset**
```bash
python code/ieso_data_prep.py
```

**3. Open the analysis notebook**
```bash
jupyter notebook code/ieso_analysis.ipynb
```

> Make sure to run all cells in order. The notebook reads from `datasets/merged_2025.csv` so step 2 must be completed first.

## Key Findings

- Ontario was a **net exporter** in 2025, exporting ~21.1 TWh vs importing ~3.1 TWh (6.9x ratio)
- Ontario flipped to net importer during only **429 hours (4.9% of the year)**
- Average demand during import hours was **21,104 MW** vs **16,391 MW** during export hours
- **65% of import hours** occurred in July and August, driven by summer cooling peaks
- The largest single import event was **3,157 MW** on August 11 at hour 13
